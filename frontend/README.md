This project is a community-driven social platform where users can create communities, participate in discussions, and interact through posts and topics. The application demonstrates fullstack development practices including authentication, REST API design, and relational database management.

Tech Stack:

Frontend:
React
TypeScript
React Router
Context API (State Management)
SCSS / Responsive UI

Backend:
Node.js
Express.js
Prisma ORM
PostgreSQL
JWT Authentication
Multer (Image Upload)

Features:
Authentication
User registration and login
JWT-based authentication
Protected routes

Community System:
Create and browse communities
Community ownership system
Community image upload
Community description and metadata

Topics & Posts:
Create discussion topics inside communities
Users can create posts inside topics
Relational data structure using Prisma

Backend Architecture:
RESTful API design
Prisma relational database modeling
Modular route structure
Middleware authentication system

Frontend Architecture:
Fully typed React components using TypeScript
Component-based UI architecture
Context API for global state management
Mobile responsive design improvements

Authentication and authorization logic

Responsive UI design
Client-server communication
